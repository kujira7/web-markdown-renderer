# Privacy Policy

Web Markdown Renderer processes selected text locally in the user's browser.

## Data Collection

This extension does not collect, store, sell, share, or transmit user data.

## Selected Text

When the user invokes the extension, it reads the selected text in the current tab and renders it as Markdown in an on-page viewer. The selected text stays in the current browser session and is not sent to any server.

## Clipboard

The `Copy source` button writes the rendered source text to the user's clipboard only after the user clicks the button.

## Network Access

This extension does not make network requests, call external APIs, use analytics, or connect to third-party services.

## Permissions

- `activeTab`: grants temporary access to the current tab after the user invokes the extension.
- `contextMenus`: adds the `Render as Markdown` item to the selection context menu.
- `scripting`: injects the renderer scripts and CSS into the current tab after user action.
